JQUERY CHEAT SHEET

1 - data types

Just like a language has nouns, verbs, and prepositions, JavaScript has its own building blocks.

There are three essential data types

NAME DESCRIPTION EXAMPLE
STRING

Any grouping of words or numbers surrounded by single quotes

'LOVE' , '5.30 in the afternoon' , 'hello world!'
NUMBER

Any number, including numbers with decimals, without quotes.

44, 8 . 1367, 23.34 (note the point)
Boolean

Boolean: This is always one of two words. Either true or false, with no quotations.

true or false
