JQUERY CHEAT SHEET

jQuery is a powerful and simple JavaScript library that you can use to select elements in your Web page, add impressive special effects, and insert content. Also find great resources for jQuery tips, tutorials, and plug-ins.

1 - call the jquery library

If you don't want to download and host jQuery yourself, you can include it from a CDN (Content Delivery Network).Both Google and Microsoft host jQuery.

Many users already have downloaded jQuery from Google or Microsoft when visiting another site. As a result, it will be loaded from cache when they visit your site, which leads to faster loading time. Also, most CDN's will make sure that once a user requests a file from it, it will be served from the server closest to them, which also leads to faster loading time.

GOOGLE

Insert the following link to a script tag. src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"
MICROSOFT

Insert the following link to a script tag. src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.12.4.min.js"
Functions In a Separate File

In order for jQuery functions to be easy to maintain accross many pages, put your jQuery functions in a separate .js file in a <script> tag.

2 - Selecting Elements with jquery.

jQuery allows you to select elements in a Web page. It's based on the existing CSS Selectors, and in addition, it has some own custom selectors.

All selectors in jQuery start with the dollar sign and parentheses: $().

for a full of jquery selectors,clickhere.

Elements Entry
Element type (for example, <img>) $(''img'')
All elements

$(''*'')
ID (for example, id=''thisIsTheID'')

$(''#thisIsTheID'')
Class (for example, class=''someClass'')

$(''.someClass'')
Order (for example, the first or last <img> element)

$(''img:first'')

$(''img:last'')
Attribute, (for example, to get the length attribute of <img>)

$(''img[height]'').length
Parent (for example, the parent of <div>)

$(''div:parent'')
Child (for example, the first or last child of <div>)

$(''div:first-child'')

$(''div:last-child'')

3 - Ready () event.

To run the jquery code as soon as the document is ready to be manipulated.

     

Jquery syntax to use with each method.

 
          $(document).ready(function(){
          ....method will go here.
        });
